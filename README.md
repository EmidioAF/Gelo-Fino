# Gelo Fino

## Integrantes
- Emidio Angelotti Filho

## Descrição
Este projeto é um jogo casual inspirado no minigame "Gelo Fino" do Club Penguin.
O objetivo é coletar os peixes no mapa e chegar na saída sem ficar preso ou cair.

## Tecnologias
- JavaScript
- p5.js
- HTML
- CSS

## Como executar
1. Baixe ou clone este repositório
2. Abra a pasta no VS Code
3. Rode o arquivo index.html com a extensão Live Server

## Controles
- W, A, S, D ou setas direcionais para mover o personagem
- ENTER para iniciar
- R para reiniciar após game over
- N para avançar para a próxima fase

## Regras do jogo
- Cada bloco de gelo quebra quando o jogador sai dele
- Peixes aumentam a pontuação
- Ao chegar na saída, o jogador vence a fase
- Se ficar sem movimentos, ocorre game over
